set_app_files {webR4Shiny}R Documentation

Get app files within the webR installation

Description

This allows to programmatically generate a list of file to write in the webR virtual file system

Usage

set_app_files(path, conn)

Arguments

path

Path containing the Shiny webR installation.

conn

webr-shiny.js content.
